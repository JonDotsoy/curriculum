Dependencies
------------

This project require [Node](https://nodejs.org/) or [iojs](https://iojs.org/) with [npm](https://npmjs.org/), [Gulp](http://gulpjs.com/) and [Bower](http://bower.io/).


Install
-------

```bash
bower install
npm install
gulp build-source
```

Deploy with Docker
------------------

Require [Docker](http://docker.io) up.

```bash
gulp build
gulp deploy
```
